# 🌤️ Weather App

A modern and responsive weather application built with **React** and **Chakra UI**.
It displays real-time weather data, a 7-day forecast, and air quality information.

---

## 🚀 Features

* 🔍 Search weather by city
* 🌡️ Temperature in Celsius / Fahrenheit
* 📅 7-Day forecast
* 🌫️ Air Quality details (PM2.5, PM10, NO₂, O₃)
* 🌙 Dark mode toggle
* 📱 Fully responsive design

---

## 🛠️ Technologies Used

* React.js
* Chakra UI
* OpenWeather API

---

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/Khawlaattou/weather-app.git
cd weather-app
```

Install dependencies:

```bash
npm install
```

Run the app:

```bash
npm start
```

---

## 🔑 Environment Variables

Create a `.env` file in the root and add:

```env
REACT_APP_WEATHER_API_KEY=your_api_key_here
```

⚠️ Do NOT share your API key publicly.
